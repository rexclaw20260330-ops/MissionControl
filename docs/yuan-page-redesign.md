# Yuan's Personal Page 改版設計文件

## 📋 設計目標

將 Yuan's Personal Page 從「技能展示型」改為「學習進度追蹤型」，強調持續學習與成長可視化。

---

## 🎯 移除項目

| 項目 | 說明 |
|------|------|
| 左下角四格 Stat Card | `Tasks Completed`, `Projects Active`, `Agent Hours`, `Win Rate` |
| Skills Radar 圖表 | 整個六邊形雷達圖移除 |
| Quarterly Goals 區塊 | 改為更動態的學習追蹤 |

---

### 間距系統

| 名稱 | 數值 |
|------|------|
| 卡片內邊距 | 24px |
| 卡片間距 | 24px |
| 區塊內元素間距 | 16px |
| 區塊內微間距 | 8px |

### 動畫效果

- 卡片 hover: `border-color` 從 `#FFD700/20` → `#FFD700/40`，持續 200ms ease
- Progress bar 填充: `transition: width 500ms cubic-bezier(0.4, 0, 0.2, 1)`
- Streak 火焰閃爍: `animation: flicker 2s infinite`
- 完成項目淡入: `opacity 0 → 1, transform translateY(10px) → 0`，300ms

## 📅 Calendar 三種視圖設計

### 共同特性

- 視圖切換器: `[日] [週] [月]` 三個標籤按鈕
- 導航: 左箭头 / 當前期間 / 右箭头
- 當日/週/月 高亮顯示
- 可點擊日期新增學習項目

## 🔧 互動行為
